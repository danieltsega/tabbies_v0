# Tabbies: Tab & Workspace Manager

Tabbies is a Google Chrome extension designed to help you organize and save your tabs in categories. By using a "Hot Storage" architecture, Tabbies can save your tabs and reopen them later **without reloading**, fully preserving their back/forward history, scroll position, and form state.

## Key Features
- **Categories**: Organize your saved tabs into custom categories with unique names, emojis, and colors.
- **Zero-Reload (Hot Storage)**: Move open tabs to a background window to declutter your workspace while keeping them alive in memory.
- **Persistent Backups (Cold Storage)**: If the browser is restarted, your links are safely backed up and can be re-opened instantly.
- **Sleek Action Popup**: Access your categories and tabs instantly from the extension toolbar.

## Installation for Developers
1. Open Google Chrome.
2. Navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked** (top-left button).
5. Select the `tabbies` directory.
